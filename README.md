# 剑指offer刷题笔记（JavaScript）

> 大厂SP冲！

题目都是`leetcode->剑指offer`的原题，文件命名规则是`题号-简短的题目描述`。
代码头部会写解法，但只是写了我认为常用的解法，不一定全面。
解法大都来源于leetcode的评论区。
本项目特色：
```
1. 会从python社区吸收优秀的解法，不只是JS社区。
2. 会尽可能多的使用JS特色的方法，比如尽量使用 *for...in...* 而不是用C语言那种模式。
3. 会标明易错点
```
